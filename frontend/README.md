# E-Commerce Web Application

A full-stack e-commerce platform built with the MERN Stack featuring user and admin roles, JWT authentication, and complete shopping flow.

Live Demo: Coming Soon
Developer: Omkar Sontakke - github.com/omkar-027

## Features
- JWT Authentication (Register / Login)
- Product Browsing, Cart and Checkout
- Order Tracking (My Orders)
- Admin Dashboard - Add, Edit, Delete Products and Manage Orders
- Role-Based Access Control (User and Admin)

## Tech Stack
Frontend: React.js, JavaScript, HTML5, CSS3, Tailwind CSS
Backend: Node.js, Express.js, MongoDB, Mongoose, JWT

## Run Locally
git clone https://github.com/omkar-027/ecommerce-mern-app.git
cd backend && npm install && npm start
cd frontend && npm install && npm start
